About Photoshop

The Interface

1. Menu bar (at the very top) : Shows the File, Edit, Image, and other menus that give you access to a variety of commands, adjustments, and panels.
2. Options bar (underneath the menu bar) : Displays options for the tool you are currently working with.
3. Tools panel (on the left) : Contains tools for editing images and creating artwork. Similar tools are grouped together. You can access related tools in a group by clicking and holding a tool in the panel.
4. Panels (on the right) : Include Color, Layers, Properties, and other panels that contain a variety of controls for working with images.
5. Document window (in the middle) : Displays the file youâ€™re currently working on. Multiple open documents show up in tabs in the Document window.
6. Close image : Choose File > Close.




Basic Tools





Move (V) : Moves selections or layers.
Marquee (M) : Makes rectangular, elliptical, single row, and single column selections.
Lasso (L) : Makes freehand, polygonal (straight-edged), and magnetic selections.
Quick Selection (W) : Make selections by painting.
Crop (C) : Crops an image.
Eyedropper (I) : Samples pixels.
